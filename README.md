# gr-ppm-wiegand
GNU Radio module for Pulse Position Modulation (PPM) Wiegand 26/29/31/33/34/36 bit RF Identification Cards (based on PPM Radio Control (RC) receiver/transmitter)

## Installing:
```
 mkdir build
 cd build
 cmake ..
 make
 sudo make install
```

## Dependencies:
- gnuradio libs/includes
- swig
- cmake
- make
- gcc/build env.
